# My Website

Welcome to the repository for **My Website**! This project showcases my portfolio as a Full Stack Developer, highlighting my skills, projects, and experiences in both frontend and backend development.

## Features

- **Responsive Design**: Fully optimized for all devices (mobile, tablet, and desktop).
- **Modern Tech Stack**: Built with React, TypeScript, and Tailwind CSS.
- **Interactive Contact Form**: Integrated with EmailJS to send messages directly to my email.
- **Dynamic Animations**: Eye-catching animations and effects for an engaging user experience.

## Tech Stack

### Frontend:

- **React**
- **TypeScript**
- **Tailwind CSS**

### Backend (Future Implementation):

- **Spring Boot**
- **PostgreSQL**
- **Node.js (for API integrations)**

## Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** (v16 or later)
- **npm** (v8 or later)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ParadelaPablo/my-website.git
   ```
2. Navigate to the project directory:

   ```bash
   cd my-website
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the development server:

   ```bash
   npm run dev
   ```
5. Open your browser and go to:

   ```
   http://localhost:3000
   ```
